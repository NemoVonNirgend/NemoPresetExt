Ember 2.0 is completely transparent to AI. Just ask the AI to:

  • "Create a chart showing sales data"

No special instructions needed! The AI writes normal code, and Ember automatically detects and runs it with all libraries available.

For Developers

Create your own asset packs using JavaScript with special metadata comments:

//@name my-dating-sim
//@display-name My Dating Sim
//@description Interactive dating simulation
//@author Your Name
//@arg relationship_status string "single"
//@arg trust_level number 0
//@context-field relationship_status string Current relationship
//@export updateRelationship

// Your JavaScript code here...
const state = { relationship: getArg('relationship_status') };
// Create UI, handle interactions, inject context
ember.inject({ content: "Updated relationship status" });

See the documentation for full API reference and examples.
