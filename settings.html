NemoPreset UI Extensions

Enhances SillyTavern's UI with better organization, visual improvements, and quality-of-life features for power users. Customize your experience below.

Feature Toggles

Core

Enable or disable individual features. Most require a page refresh.

Core Features

Prompt Manager Enhancements
Collapsible sections, drag-and-drop reordering, and visual improvements
Section Dropdown Style
Choose how prompt sections expand. Tray Overlay shows a floating panel; Accordion List expands prompts inline with drag-and-drop reordering.
Preset Navigator
Browse, search, and manage API presets with the "Browse..." button
Prompt Directive System
Allows preset makers to embed rules and dependencies in prompts
Directive Autocomplete
Show autocomplete suggestions when typing directives in the prompt editor Requires refresh

Optional UI Features

Animated Backgrounds
Support for video backgrounds (webm, mp4) and YouTube URLs with controls Requires refresh
Settings Tab Overhauls
Tabbed interface for User Settings and Advanced Formatting Requires refresh
Extensions Tab Overhaul
Organized categories with search for extensions Requires refresh
Lorebook UI Overhaul
Enhanced Lorebook interface with improved organization Requires refresh
Unified Reasoning Section
Consolidated reasoning controls in the prompt manager
Quick Lorebook Access
Manage active lorebooks directly from the prompt manager
Auto-Trim Old HTML
Automatically convert complex HTML/CSS in old messages to simple text dropdowns to reduce context usage
Wide Navigation Panels
Make left and right navigation panels take up 50% of screen width Requires refresh
Mobile UI Enhancements
Optimized layouts, larger touch targets, and improved readability on touch devices Auto-detects touch devices
Pollinations Image Interceptor
Intercept Pollinations.ai image URLs in messages and regenerate them using SillyTavern's configured image generation backend Uses your SD/DALL-E/etc settings
